LE PROJET

Spoti-Stats est un dashboard interactif permettant aux utilisateurs de visualiser leurs habitudes d'écoute Spotify. Le projet utilise l'API Web de Spotify pour récupérer des données en temps réel telles que les artistes les plus écoutés, les titres préférés et les genres dominants. Use OAuth 2.0 pour une authentification sécurisée.

TECH STACK

  • Frontend : HTML5, CSS3, JavaScript (ES6+).
  • Data Viz : Chart.js (ou équivalent custom).

INTERFACE

TABLEAU DE BORD

Une vue d'ensemble claire et esthétique des statistiques clés. Le design sombre (Dark Mode) reprend les codes visuels de Spotify tout en y ajoutant une touche personnelle avec des gradients néons.

ACCUEIL & CONNEXION

Page d'atterrissage incitant l'utilisateur à se connecter via son compte Spotify. L'expérience est fluide et sécurisée grâce au protocole d'autorisation standard.
