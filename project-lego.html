LE PROJET

LEGO Manager est un projet réalisé dans le cadre de la SAE 203. L'objectif était de développer un site web dynamique permettant aux utilisateurs de gérer leur propre collection de sets LEGO.

Base de données. Espace Membre. Interactivité.

Le site a été entièrement développé en PHP natif sans framework, avec une base de données SQLite. Il intègre de nombreuses fonctionnalités comme l'import de données CSV, un système de commentaires, une wishlist et un chat en temps réel.

FONCTIONNALITÉS

  • Collection : Ajout/Retrait de sets, Wishlist personnelle.
  • Communauté : Commentaires, notes et Chat membres.
  • Back-office : Import CSV, gestion des utilisateurs (Admin).
  • Sécurité : Hashage des mots de passe, validation email.
