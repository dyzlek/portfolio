LE CHALLENGE

Le Marathon MMI 2026 est une semaine intensive où les étudiants de 2ème année MMI doivent créer une identité complète autour d'un sujet imposé.

4 jours de production. Une équipe de 6. Un résultat concret.

Dans ce projet "Arlett", j'étais en charge du développement web et de l'intégration. Le défi était de traduire la direction artistique de mes coéquipiers en un site WordPress fonctionnel et responsive dans un temps très court.

TECHNIQUES

  • WordPress : CMS pour la structure du site.
  • Elementor : Builder pour la mise en page avancée.
  • CSS Personnalisé : Ajustements fins pour coller à la DA.
  • Travail d'équipe : Collaboration étroite avec les designers.
