LE PROJET

La SAE 203 est un projet universitaire visant à concevoir et développer un site web dynamique utilisant PHP et une base de données MySQL.

Un catalogue interactif pour les passionnés de LEGO.

J'ai développé une application complète permettant aux utilisateurs de s'inscrire, gérer leur propre collection de sets LEGO, ajouter des items à leur wishlist et interagir avec la communauté via des commentaires et des notes.

TECHNIQUES

  • PHP / MySQL : Backend robuste et gestion de base de données.
  • Architecture MVC : Structure de code propre et maintenable.
  • Sécurité : Hashage de mots de passe, protection XSS/SQL Injection.
  • Bootstrap : Interface responsive et moderne.
